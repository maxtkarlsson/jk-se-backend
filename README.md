https://jk-se-backend.vercel.app/api/v1/products

## Project setup

To install all dependencies, run:

```
npm i
```

Create an .env file. See example file in project.

To seed the database with products, run:

```
npm run seedDb
```

## Run project

To start the project, run:

```
npm run dev
```

It will run on:

```
http://localhost:5173/
```
